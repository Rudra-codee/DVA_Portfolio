## ✨ Features

- **GitHub Dark Theme** — pixel-accurate GitHub dark mode aesthetic
- **Project Cards** — filterable repository cards with cover images, tags, and modal previews
- **Live GitHub Snake** — real contribution graph rendered as an animated snake (auto-updates daily via GitHub Actions)
- **Search & Filter** — find projects by name or filter by category (Python · AI / Agents · Space)
- **Currently Building Banner** — live status of what I'm actively working on
- **Responsive Design** — works on desktop and mobile
- **Modal Popups** — click any card for a full project description and GitHub link

## 🚀 Getting Started

### Run Locally

```bash
# Clone the repo
git clone https://github.com/Rudra-codee/portfolio.git

# Open in browser
open index.html
```

No build step required — pure HTML, CSS, and JavaScript.
